# Fudami — Immersive Japanese Mastery

Fudami is a modern, cross-platform language learning ecosystem designed for peak memory retention and deep structural understanding of Japanese. 

This repository contains the **Marketing & SEO Landing Page** for the Fudami platform.

---

## 🚀 The Fudami Ecosystem

Fudami is built on a 4-pillar architecture to ensure professional-grade performance, SEO, and security.

| Component | Role | URL | Visibility |
| :--- | :--- | :--- | :--- |
| **Landing Page** | **Marketing & SEO** | [fudami.com (Demo)](https://snonow.github.io/fudami-landing/) | **Public** |
| **Web App** | **Client Application** | [app.fudami.pages.dev](https://fudami.pages.dev) | **Public** |
| **Backend** | **API & Content Delivery** | Internal | **Private** |
| **Studio** | **Content Generation** | Internal | **Private** |

---

## 🛠 Features

- **SEO Optimized**: High-performance static HTML for maximum search engine visibility.
- **Responsive Design**: Tailored for both desktop and mobile discovery.
- **Conversion-Focused**: Direct links to the web app and future mobile store listings.

---

## 🏗 Development

This is a static site. No complex build step is required for local previewing.

1. Clone the repository.
2. Open `index.html` in any browser.

### Deployment
Automated via **GitHub Actions**. Pushing to the `main` branch triggers a deployment to GitHub Pages.
